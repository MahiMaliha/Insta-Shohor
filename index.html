** What are the differences between local and session storage?

*We can access local storage from any window we want.But if we want to access session storage,then we need to access it from the same window.

*In local storage,the storage capacity is 5mb or 10 mb,it varies with the browser.But in session storage,the maximum capacity of the storage is 5 mb.

*We can delete local storage after a fixed time.But session storage can be deleted when the window is closed properly.

*To save an item in local storage we write (localStorage.setItem('Hi','how are you')). In terms of session storage we write (sessionStorage.setItem('Hi','how are you'))

** What Does javascript event loop do?

Basically javascript event loop works like for/while loop & also priority base like which event come first.Generally event loop firstly take data or event from stact then run it.If event loop cannot find any data or event from stact then it takes data from queue.The event loop is a constantly running process that monitors both the callback queue and the call stack.If the call stack is not empty, the event loop waits until it is empty, and places the next function from the callback queue to the call stack.If the callback queue is empty, nothing will happen.
